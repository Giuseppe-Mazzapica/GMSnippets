== Description ==

GM Snippets is a plugin for easy managment of reusable content and gives the ability to reuse content via shortcode, widget or template tag

= Minimum Requirements =

* WordPress 3.5 or greater
* PHP version 5.3 or greater
* MySQL version 5.0 or greater
